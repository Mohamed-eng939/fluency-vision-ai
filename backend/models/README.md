
# Acoustic Models Directory

This directory should contain your MFA acoustic models and dictionaries:

1. Place the English dictionary file here, named `english_us_dict.dict`
2. Place the English acoustic model directory here, named `english_us_mfa` 

You can download these files from the MFA website or use the MFA CLI to download them:

```bash
mfa model download acoustic english_us_mfa
mfa model download dictionary english_us_dict
```
